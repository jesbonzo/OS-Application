Scheduling Algorithm

Preemptive Priority Scheduling Algorithm (PPSA)

Preemptive

  • Preemptive Priority Scheduling Algorithm is the preemptive version of the Priority Scheduling Algorithm, but if a new process having a higher priority than the currently running process arrives, it gets selected immediately.
  • The new process has not to wait until the currently running process finishes or yields.
